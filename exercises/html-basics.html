Paragraphs and Text Formatting

This is a regular paragraph. Paragraphs are the most common way to display text content.

You can make text bold/important or italic/emphasized.

There's also highlighted text, small text, and deleted text.

For code, use inline code or:


                // Multi-line code block
                function hello() {
                    return "Hello, World!";
                }
